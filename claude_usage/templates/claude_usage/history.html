Tool Glossary

File Operations

Read
Reads a file from the filesystem. Used to inspect code, configs, or any text file.
Edit
Makes a precise string replacement inside an existing file. Preferred over Write for partial changes.
Write
Creates a new file or completely overwrites an existing one.
Glob
Finds files by name pattern (e.g. **/*.py). Returns paths sorted by modification time.
Grep
Searches file contents using regex (ripgrep). Used to find symbols, strings, or patterns across the codebase.
Notebook
Edit
Edits cells in a Jupyter notebook (.ipynb file).

Execution

Bash
Runs a shell command. Used for builds, tests, git, package managers, and anything CLI.

Research

Web
Fetch
Fetches and reads a web page or URL content.
Web
Search
Performs a web search and returns results.

Agents & Tasks

Agent
Spawns a sub-agent (Explore, Plan, general-purpose, etc.) to handle a task in an isolated context.
Todo
Write
Manages the session task checklist — creates, updates, or completes to-do items.
Task
Create
Creates a tracked task. Part of the extended task-management tool family (TaskGet, TaskUpdate, TaskStop, etc.).

MCP Servers

mcp__*
Calls a tool on an MCP (Model Context Protocol) server. The suffix identifies the server and tool, e.g. mcp__ide__getDiagnostics.
